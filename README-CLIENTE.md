# Aplicación Thinkia - Softphone AI Contact Experience

## ¿Qué es esto?

Esta es una aplicación web completamente funcional desarrollada con React y Tailwind CSS.
**NO es un archivo de Figma** - es código real que funciona como una aplicación web.

## Para visualizar la aplicación

### Opción A: Ver online (más fácil)
Si el desarrollador desplegó la aplicación, simplemente abre el enlace que te compartió.

### Opción B: Ver localmente en tu computadora

**Requisitos:**
- Node.js 18 o superior instalado (descargar de https://nodejs.org)
- pnpm instalado (`npm install -g pnpm`)

**Pasos:**
1. Descomprime el archivo ZIP
2. Abre una terminal en la carpeta del proyecto
3. Ejecuta:
   ```bash
   pnpm install
   pnpm run dev
   ```
4. Abre tu navegador en http://localhost:5173

## Rutas de la aplicación

- `/` - Login
- `/dashboard` - Centro de llamadas
- `/call-history` - Historial de llamadas
- `/numbers` - Configuración de números
- `/teams` - Equipos
- `/monitoring` - Monitoreo en vivo
- `/analytics` - Analíticas
- `/team-directory` - Directorio de equipo
- `/settings` - Configuración
