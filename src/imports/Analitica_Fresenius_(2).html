ℹ️ ¿Cómo se calcula el Tiempo de Resolución?
Tiempo de resolución = Tiempo en llamada + Tiempo de trabajo posterior (ACW — After Call Work).
El ACW incluye el tiempo que el agente dedica a documentar, clasificar o gestionar tareas derivadas tras colgar. Se considera resuelto cuando el agente vuelve al estado "disponible".
Fórmula: t_resolucion = t_llamada + t_ACW — Objetivo: < 3 min.
